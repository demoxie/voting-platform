API Documentation

1. Authentication API

Base URL: /api/v1/auth

GET /api/v1/auth/login
POST /api/v1/auth/logout
POST /api/v1/auth/register

Authentication endpoints for user login, logout, and registration.

2. User API

Base URL: /api/v1/users

GET /api/v1/users
GET /api/v1/users/{id}
POST /api/v1/users
PUT /api/v1/users/{id}
DELETE /api/v1/users/{id}

Endpoints for managing users. Includes operations to list, retrieve, create, update, and delete users.

3. Voting API

Base URL: /api/v1/votings

GET /api/v1/votings
GET /api/v1/votings/{id}
POST /api/v1/votings
PUT /api/v1/votings/{id}
DELETE /api/v1/votings/{id}

Endpoints for managing votes. Includes operations to list, retrieve, create, update, and delete votes.
